# Security Policy

ClickGuide Local records browser workflows and stores screenshots locally. Security and privacy issues should be handled carefully.

## Supported Versions

The current `main` branch is the supported development version.

## Reporting A Vulnerability

Please report security issues through GitHub Security Advisories when available. If advisories are not enabled, open a GitHub issue with a minimal description and avoid posting sensitive screenshots, tokens, credentials, or private URLs.

## Privacy Expectations

- The extension should not transmit captured screenshots, URLs, notes, or click data to external services.
- Password fields and elements marked with `data-clickguide-ignore` should not be recorded as click targets.
- Permission changes must be documented in `README.md`.
